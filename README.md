# 🏦 OOP Console Bank System

A TypeScript/Node.js CLI-based banking application built using Object-Oriented Programming (OOP) principles. It simulates real-world banking operations directly in the terminal with a colorful, interactive user interface.

---

## ✨ Features

* **Interactive CLI Menu:** Smooth navigation using `inquirer`.
* **Mock Data Generation:** Automatically populates random customer profiles using `@faker-js/faker`.
* **Account Operations:**
  * View current balance with personalized greetings.
  * Secure cash withdrawal with instant balance validation.
  * Real-time cash deposit updating the system balance.
* **Styled Console Output:** Clean layout with distinct neon text colors using `chalk`.

---

## 🛠️ Tech Stack

* **Language:** TypeScript / Node.js
* **Dependencies:** `inquirer` (User Prompts), `chalk` (Terminal Styling), `@faker-js/faker` (Mock Data generation)
